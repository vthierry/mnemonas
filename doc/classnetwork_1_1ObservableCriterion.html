Defines the estimator criterion.

  • The method is to be overwritten to implement a global estimator.
  • The default implementation simply returns , running:
    double r = 0;
transform.reset(true);
for(unsigned int t = 0; t < transform.T; t++)
  for(int n =  transform.N - 1; 0 <= n; n--)
     r += rho(n, t, transform.get(n, t));
return r / (transform.N *  transform.T);

    The criterion value to mnimize.
