Resets the criterion for a given transform.

Parameters
transformThe transform to estimate.
valuesThe output desired values.
  • By contract output units correspond to the values.N lower indexed units and hidden units to the remainder indexes.
criterionThe criterion index:
  • 2 The L2 criterion .
  • 1 The reweighted L1 criterion .
  • 0 The reweighted L0 criterion .
  • a A L1 related unbounded criterion .
    • If simply considers the absolute value of the error.
  • b A L0 related biweight bounded criterion .
    • If simply counts the number of non-zero values.
  • h The L2 boolean criterion .
    • If simply counts the number of wrong boolean values (not used as criterion but a posteriori precision).
nuThe criterion margin parameter.
reinjectDefined how desired values are defined:
  • 'n' : output values are not reinjected as desired values.
  • 'o : output values are reinjected as desired values, other values are not used.
  • 'b' : output values are reinjected and backpropagated on hidden values in order to match the output values.
