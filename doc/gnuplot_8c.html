Generates a gnuplot dipslay script.

  • Generates a script to display gnuplot commands.
    Parameters
    fileThe related data file name (without extension). By contract,
    • the file basename is used as the plot title;
    • the file name is used to generate a .gnuplot.sh display script.
    • the file name is used to generate a .png image of the plot.
    • the data is stored in a file of extension .dat.
    plotThe gnuplot display commands.
    • The $title variable contains the plot title "plot-title" parameter. See: gnuplot set commands.
    showIf true performs on the fly display.
